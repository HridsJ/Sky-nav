Some details about SKYnav

Skynav is a web application designed to optimize flight paths for small aircraft, including drones and light helicopters. Leveraging the OpenWeather API, it gathers real-time data on snow, rain, and wind. A sophisticated algorithm, combining A* search with a scikit-learn forest classifier, calculates the safest, most efficient routes, minimizing weather exposure. These routes are then visualized on an interactive map using Leaflet, providing pilots with clear, dynamic guidance. Skynav enhances flight safety and efficiency by delivering data-driven, weather-aware route planning within a user-friendly web interface. This accessibility allows for pre-flight planning and in-flight adjustments from any device with internet access.





FUTURE ENDEAVOURS

Skynav aims to expand its capabilities by integrating more comprehensive weather data, including icing, turbulence, and visibility. Future development includes real-time air traffic data integration for collision avoidance and dynamic rerouting. Machine learning will be employed to refine the forest classifier for more accurate weather impact predictions. Implementing 3D mapping and terrain analysis will enhance route visualization and safety. Skynav will also explore integration with onboard flight systems for seamless data transfer and automated flight adjustments. User-specific profiles and preferences will be incorporated for personalized route optimization, aiming to become a complete flight planning and navigation solution.
